About the course

Learn by Building

From baselines to production-ready pipelines

This workshop takes you from classical NLP baselines to cutting-edge LLM applications in three days. Every concept is taught through runnable code in Google Colab — no local setup required.

You’ll work with real datasets, real APIs (Groq, Together.AI), and real evaluation frameworks. By the end, you’ll have a working NLP pipeline and the judgment to choose the right tool for any text analysis task.

  1. Build and evaluate TF-IDF, embedding, and LLM-based classifiers
  2. Extract structured data from unstructured text using Pydantic schemas
  3. Discover topics with BERTopic and annotate them with LLMs
  4. Build semantic search with FAISS and cross-encoder reranking
  5. Fine-tune a small language model (Qwen3-4B) using LoRA
  6. Evaluate models rigorously: metrics, slicing, inter-rater reliability

Setup Guide

Everything is free — 10 minutes to get ready
Required

Google Colab

All notebooks run in Colab — no local install needed. Just sign in with your Google account.

  1. Go to colab.research.google.com
  2. Sign in with your Google account
  3. Verify GPU access: Runtime → Change runtime type → T4 GPU
Required

Groq API Key (Free)

Our primary LLM provider. Extremely fast inference, generous free tier (14,400 requests/day).

  1. Go to console.groq.com/keys
  2. Create a free account (GitHub or Google sign-in)
  3. Click “Create API Key”
  4. Save the key — you’ll paste it into each notebook
Recommended

Together.AI API Key ($5 free credit)

Backup provider with wider model selection. Useful if Groq rate limits are hit during class.

  1. Go to api.together.xyz
  2. Create account → get $5 free credit
  3. Copy your API key
Optional

Clone the Repository

For offline access or running notebooks locally. Not required — Colab links work directly.

git clone https://github.com/RJuro/unistra-nlp2026.git
