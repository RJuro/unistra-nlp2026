Setup Guide

Everything is free. Takes about 10 minutes.

Required

Google Colab

All notebooks run in Colab — no local install needed.

  1. Go to colab.research.google.com
  2. Sign in with your Google account
  3. Verify GPU: Runtime → Change runtime type → T4 GPU
Required

Groq API Key (Free)

Primary LLM provider. Fast inference, 14,400 requests/day free.

  1. Go to console.groq.com/keys
  2. Create a free account (GitHub or Google sign-in)
  3. Click “Create API Key” and save it
Recommended

Together.AI API Key

Backup provider with $5 free credit. Useful if Groq rate limits are hit.

  1. Go to api.together.xyz
  2. Create account → get $5 free credit
  3. Copy your API key
Optional

Clone the Repository

For offline access. Not required — Colab links work directly.

git clone https://github.com/RJuro/unistra-nlp2026.git

Project Tracks

Pick one track, work across Sprint 1 (Tue) and Sprint 2 (Thu). Own data welcome.
