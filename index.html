Background

Concept Explainers

Short animated videos covering key concepts we build on in the workshop.

Word2Vec

How words become vectors — the skip-gram model, training on context windows, and why "king − man + woman = queen" works.

Self-Attention

The mechanism at the heart of Transformers — queries, keys, values, and how attention lets models understand context.

LDA Topic Modeling

Latent Dirichlet Allocation — how documents are modeled as mixtures of topics, and topics as distributions over words.

LoRA Fine-tuning

Low-Rank Adaptation — how freezing model weights and training small adapter matrices makes fine-tuning feasible on free hardware.

Setup Guide

Everything is free. Takes about 10 minutes.

Required

Google Colab

All notebooks run in Colab — no local install needed.

  1. Go to colab.research.google.com
  2. Sign in with your Google account
  3. Verify GPU: Runtime → Change runtime type → T4 GPU
Required

Groq API Key (Free)

Primary LLM provider. Fast inference, 14,400 requests/day free.

  1. Go to console.groq.com/keys
  2. Create a free account (GitHub or Google sign-in)
  3. Click “Create API Key” and save it
Recommended

Together.AI API Key

Backup provider with $5 free credit. Useful if Groq rate limits are hit.

  1. Go to api.together.xyz
  2. Create account → get $5 free credit
  3. Copy your API key
Optional

Clone the Repository

For offline access. Not required — Colab links work directly.

git clone https://github.com/RJuro/unistra-nlp2026.git

About This Workshop

Applied NLP: From Text to Intelligence

A 20-hour intensive workshop that takes social scientists from zero NLP to building production-quality text analysis pipelines — all in three days.

When
February 10–12, 2026
Where
Room A 330, Idem-Lab, University of Strasbourg
Instructor
Roman Jurowetzki — Aalborg University / University of Strasbourg
Format
5 blocks × 4 hours — lectures, guided notebooks, project sprints

What You'll Learn

The workshop covers the full modern NLP pipeline, from classical baselines to cutting-edge LLM techniques:

  • Day 1 (Tue): TF-IDF baselines, sentence embeddings, LLM zero-shot classification, BERTopic, project kick-off
  • Day 2 (Wed): SetFit few-shot learning, FAISS semantic search, evaluation best practices
  • Day 3 (Thu): Cross-encoder reranking, LLM distillation, LoRA fine-tuning, annotation & IRR, final presentations

Learning Outcomes

  • Build text classification pipelines from TF-IDF to fine-tuned transformers
  • Use LLMs for zero-shot classification with structured JSON output
  • Discover topics in large corpora with BERTopic + LLM annotation
  • Build semantic search systems with FAISS and cross-encoder reranking
  • Generate training data with LLMs and fine-tune small models via LoRA
  • Evaluate models rigorously: macro F1, slice analysis, inter-rater reliability

Prerequisites

Basic Python familiarity (loops, functions, pandas). No NLP or ML experience required — we start from scratch. Everything runs in Google Colab with free API keys.
